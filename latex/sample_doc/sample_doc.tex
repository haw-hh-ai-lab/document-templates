\subsection{Das Problem}
\label{ssec:intro:problem}

Aus einer Reihe von Bereichen der IT kommen heute Forderungen nach
verbesserten Storage-Systemen. Vor allem Konzepte wie \emph{Ubiqutous
Computing} mit seiner Vielzahl von allgegenwärtigen intelligenten
Geräten, \emph{Grid Computing} mit domänenübergreifenden
Anwendungen und \emph{Cluster Computing} mit einer enormen Anzahl an
Rechner-Knoten verlangen nach Verbesserungen bei

\begin{itemize}
\item Ad-Hoc Konfiguration/Discovery
\item Skalierbarkeit
\item (scheinbarer) Stabilität
\item Organisatorische Transparenz
\end{itemize}
